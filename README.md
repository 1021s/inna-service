# ZHome - Home Listing Application
> Photo Modal Carousel

![](ZhomeReadMe.png)

To build, run the command:
```sh
npm run build
```

If you want to have watch set up with your build, run the command:
```sh
npm run watch
```

To start the server:
```sh
npm start
```

To run tests, run the command:
```sh
npm test
```
